# 🎤 Offline Voice Command Deck

I built this project to automate common Windows tasks using both text and offline voice commands. The application can launch Windows applications, open websites, and maintain a history of executed commands through a simple and user-friendly interface.

## 🚀 Features

- Offline Voice Commands
- Text Command Execution
- Windows Application Automation
- Open Websites
- Custom Command Support
- Command History

## 🛠️ Tech Stack

- Python
- Streamlit
- Faster Whisper
- SoundDevice
- NumPy
- SciPy
- JSON

## 📌 Supported Commands

- Open Calculator
- Open Notepad
- Open Explorer
- Open GitHub
- Open YouTube
- Initialize Workspace
